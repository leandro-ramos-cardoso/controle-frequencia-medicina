## Navegação
- **Mobile:** barra inferior fixa (Início · Registrar · Histórico · Solicitações ·
  Perfil), botão "Registrar" destacado no centro.
- **Desktop:** sidebar recolhível + cabeçalho.
- Botão principal de ponto muda de estado conforme situação: Registrar entrada /
  Registrar saída / Registrar intervalo / Retornar do intervalo / Registro já
  realizado / Fora do horário permitido.
